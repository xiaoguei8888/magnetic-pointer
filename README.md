# Magnetic Pointer - 磁力吸附光标

## 概念

将自定义磁力光标注入任意网页。光标由两个独立部分组成：

| 元素           | 说明                                     |
| -------------- | ---------------------------------------- |
| **四角框选框** | 绿色角标组成的方框，可自转、磁力吸附     |
| **中心圆点**   | 始终在鼠标物理坐标上的圆点（独立于框选） |

框选框是鼠标位置的**附加视觉效果**，圆点是鼠标物理位置的**精确指示**。两者解耦设计，互不干扰。

## 功能

### 鼠标跟随

- 非 hover 时：框选框跟随鼠标移动（无阻尼），可配置慢速自转 + 缩放呼吸
- hover 到可交互元素时：框选框扩大框住目标，触发磁力吸附（90% 阻尼），自转暂停
- 中心圆点始终在鼠标物理坐标上（不受框选尺寸、CSS 过渡、亚像素影响）

### 可交互元素检测

自动识别以下元素（事件委托，兼容动态 DOM）：

- 原生标签：`<a>`、`<button>`、`<input>`、`<select>`、`<textarea>`、`<summary>`
- 属性标记：`[onclick]`、`[href]`、`[data-href]`、`role="button/link/menuitem/tab"`
- CSS 兜底：`cursor: pointer` 的元素
- 自动跳过内联文本标签（`<span>`、`<em>` 等），向上查找真正的交互容器
- 自动跳过视频播放器区域，保留播放器原生光标

### 防抖与稳定性

| 机制           | 说明                                                                            |
| -------------- | ------------------------------------------------------------------------------- |
| 离开延迟 120ms | hover 边缘时防止出入反复触发抖动：离开时延迟 120ms 再脱离，若期间重新进入则取消 |
| 圆点定位       | 独立 `position:fixed` 元素，通过 CSS 变量 `--x`/`--y` 驱动，零重排、零测量      |
| 框选居中       | `position:absolute` + `top/left:calc(-50%)` 相对于定位包裹层，不依赖 margin     |

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 打开扩展 → 添加新脚本 → 粘贴 `magnetic-pointer.user.js` 全部内容
3. 保存后默认在所有网页生效
4. 设置通过 `GM_setValue` 全局存储（跨域名），首次自动从旧版 `localStorage` 迁移

## 站点匹配

默认匹配所有 http/https 站点。限定站点时修改脚本头部的 `@match`：

```
// @match        https://example.com/*
```

## 操作入口

| 入口                 | 说明                                                     |
| -------------------- | -------------------------------------------------------- |
| 左上角 ⚡ 半透明角标 | 点击打开设置面板（仅在脚本启用时显示，颜色随光标色联动） |
| `Alt + Shift + M`    | 全局开关                                                 |
| `Alt + Shift + ,`    | 打开设置面板                                             |

快捷键可在 `DEFAULTS` 对象中修改（设为 `''` 禁用）。

## 设置面板

`Alt + Shift + ,` 或点击左上角 ⚡ 打开。所有改动即时生效并全局持久化。

| 设置项   | 说明                                                           | 默认值    |
| -------- | -------------------------------------------------------------- | --------- |
| 光标颜色 | 角标边框 + 中心圆点颜色，右侧 ↺ 一键恢复默认                   | `#17f700` |
| 角标粗细 | 四角边框宽度（rem）                                            | `0.20`    |
| 中心圆点 | 在鼠标位置显示指示圆点                                         | 开        |
| 慢速自转 | 非 hover 时框选框缓慢旋转（10 秒一圈）                         | 开        |
| 缩放呼吸 | 框选框大小变化（宽度/高度动画，角标粗细不变，4 秒一周期）      | 开        |
| 覆盖光标 | 隐藏原生光标，用磁力效果替代。关闭后原生光标与磁力效果叠加显示 | 开        |
| 空闲恢复 | 非 hover 时恢复原生光标（仅「覆盖光标」开启时生效）            | 关        |

## 架构

```
DOM 结构:
  <html>                         ← .mag-pointer-active 控制 cursor:none
  ├── .mag-pointer-wrap          ← translate(dampedX, dampedY)，框选定位
  │   └── .mag-pointer           ← 尺寸、自转动画、四角子元素
  │       ├── .mag-pointer__corner--tl / --tr / --bl / --br
  ├── .mag-dot                   ← translate(calc(var(--x)-50%), calc(var(--y)-50%))，独立圆点
  ├── .mag-badge                 ← 左上角设置入口
  └── .mag-settings-overlay      ← 设置面板

事件流:
  mousemove  →  setPosition(damped/raw) + updateCenterDot(raw mouse)
  mouseover  →  findInteractiveAncestor(向上遍历 DOM) → attachToElement
  边界离开   →  scheduleLeave(120ms 延迟) → 验证后 detachFromElement
  scroll     →  重新计算阻尼位置 + 边界检测
  keydown    →  快捷键匹配 (e.code)
```

## 高级自定义

编辑脚本顶部 `DEFAULTS` 对象（不通过设置面板暴露的参数）：

| 参数            | 说明                               | 默认值        |
| --------------- | ---------------------------------- | ------------- |
| `damping`       | 磁力阻尼系数（0~1，越小吸附越强）  | `0.1`         |
| `paddingRatio`  | 框选时额外内边距 = 视口宽度 × 此比 | `1/50`        |
| `rotateSpeed`   | 自转一圈的秒数                     | `10`          |
| `pulseSpeed`    | 缩放呼吸一周期秒数                 | `4`           |
| `centerDotSize` | 圆点直径（rem）                    | `0.4`         |
| `defaultSize`   | 框选框默认尺寸（rem）              | `4`           |
| `cornerSize`    | 角标线段长度（rem）                | `1`           |
| `zIndex`        | 光标层级                           | `2147483647`  |
| `cursorExclude` | 不隐藏原生光标的 CSS 选择器列表    | 视频播放器    |
| `toggleKey`     | 开关快捷键                         | `Alt+Shift+M` |
| `settingsKey`   | 设置面板快捷键                     | `Alt+Shift+,` |

## 性能

| 操作                             | 频率              | 开销     | 说明                                                            |
| -------------------------------- | ----------------- | -------- | --------------------------------------------------------------- |
| `mousemove` 位置更新             | ~60-125 Hz        | **极低** | `transform: translate()` 仅触发合成器，无布局/绘制              |
| 圆点位置更新                     | ~60-125 Hz        | **极低** | `setProperty(--x/--y)` 仅样式失效，零布局                       |
| `mousemove` 边界检测             | hover 期间        | **低**   | `getBoundingClientRect()` 触发同步布局，单次 <0.3ms             |
| `mouseover` 元素检测             | 进入新元素时      | **中**   | 沿 DOM 向上遍历，最坏 O(depth)；`getComputedStyle` 触发样式重算 |
| 框选尺寸过渡                     | hover 进入/离开时 | **中**   | `width`/`height`/`top`/`left` 0.2s 过渡触发布局；完成后开销为零 |
| 自转动画 (`rotate`)              | 非 hover 期间     | **极低** | GPU 合成，不触发布局                                            |
| 缩放呼吸 (`width`/`height` 动画) | 非 hover 期间     | **低**   | 触发布局，但仅 4 个角标子元素，重绘范围极小                     |
| `scroll` 事件                    | 滚动时            | **低**   | 仅 hover 期间执行，与 mousemove 逻辑相同                        |
| 全局 CSS `cursor: none`          | 始终              | **极低** | 一条样式规则，无运行时开销                                      |

**总结**：正常使用下 CPU 占用可忽略，GPU 合成层处理绝大部分动画。唯一的布局触发点（hover 边界检测和尺寸过渡）仅在鼠标进出可交互元素时发生，频率低且持续时间短（0.2s）。未使用 `requestAnimationFrame` 轮询，所有逻辑由浏览器事件驱动。
